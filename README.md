# ML From Scratch

Machine learning algorithms implemented from scratch using only NumPy — no shortcuts, no black boxes.

Built to develop deep understanding of how ML algorithms work under the hood, with each implementation validated against scikit-learn.

---

## Algorithms

| Algorithm | Description | Dataset |
|---|---|---|
| [Linear Regression](./linear-regression/) | Simple & Multiple Linear Regression with gradient descent | Salary, Housing |
| [Logistic Regression](./logistic-regression/) | Binary classification with L1 & L2 regularization | Breast Cancer |
| [Neural Network](./neural-network/) | Two-layer neural network built from scratch | MNIST Digits |
| [Decision Trees](./decision-trees/) | Decision tree classifier from scratch | - |
| [K-Means Clustering](./kmeans/) | Unsupervised clustering algorithm from scratch | - |

---

## What Makes This Different

Every algorithm is:
- Built from scratch using only NumPy
- Debugged and fixed when things broke (gradient explosion, shape mismatches)
- Validated against sklearn to confirm correctness
